# Criar publicação para o LinkedIn

Crie uma publicação baseada exclusivamente no trabalho realizado hoje no projeto Code Ink.

Antes de escrever:

1. Leia o arquivo `AGENTS.md`.
2. Leia `docs/LEARNING_LOG.md`.
3. Consulte as alterações atuais do Git.
4. Consulte os commits recentes, quando necessário.
5. Não invente experiências, conhecimentos, empresas ou resultados.

Caso não seja possível identificar claramente o que foi feito hoje, pergunte antes de criar a publicação.

A publicação deve:

- explicar o que foi desenvolvido;
- destacar o que aprendi;
- mostrar uma dificuldade real, quando existir;
- explicar por que a melhoria foi importante;
- usar tom profissional, natural e humilde;
- evitar exageros;
- evitar frases motivacionais genéricas;
- não afirmar que domino uma tecnologia apenas por ter feito um exercício.
